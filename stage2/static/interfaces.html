MONITORING ARCHITECTURE NOTE
The SHIELD Gateway operates in Transparent Bridge/Tap Mode.
Stage 1 sniffs raw frames using zero-copy libpcap ring buffers directly from active network device interfaces.
To change the primary interface under inspection, update the --interface argument inside /etc/systemd/system/ddos-stage1.service and restart the daemon.
